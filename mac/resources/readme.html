Installation Information

System Checks

The following checks will be performed before installation:

✓ Checking system platform and architecture
✓ At least 100MB of free space required

Installation Details

  • Install command-line tools to /usr/local/bin/
  • Generate a default configuration file with VPN destinations
  • Set proper permissions for binaries and configuration files
  • Check for WireGuard tools (required for VPN functionality)

Installation Locations

  • /usr/local/bin/gnosis_vpn - Main VPN client daemon
  • /usr/local/bin/gnosis_vpn-ctl - Control utility
  • /etc/gnosisvpn/config.toml - Configuration file
